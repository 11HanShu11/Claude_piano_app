# おんぷノート 🎹

ピアノ初学者向けの練習手帳アプリです。インストール不要のWebアプリ（PWA）として、iPhoneのホーム画面から普通のアプリのように使えます。

## できること

### 練習メニュー
練習タブを開くとカテゴリを選ぶメニューが表示され、選んだ練習が始まります。「← 選び直す」でいつでもメニューに戻れます。

- **音階読み（白鍵）**
  五線譜に表示された音符をイタリア式（ド・レ・ミ・ファ・ソ・ラ・シ）で答える4択クイズ。ト音記号／ヘ音記号を切り替えられ、「基本」「ひろめ」で音域も調整できます。正解すると音が鳴り、鍵盤のどの位置かが光ります。
- **音階読み（♯あり）**
  白鍵に加えて黒鍵の音（ド♯・レ♯・ファ♯・ソ♯・ラ♯）も出題されます。異名同音は避け、♯表記に統一しています。
- **音符の種類**
  五線譜に表示された音符が、全音符・二分音符・四分音符・八分音符のどれかを当てます。
- **調・調号**
  五線譜に表示された調号から、ハ長調・ト長調・ニ長調・イ長調・ヘ長調・変ロ長調・変ホ長調のどれかを当てます。
- **耳で音当て**（絶対音感の入り口）
  鳴った音がドレミのどれかを聴いて当てます。「もう一度きく」で何度でも聴き直せます。
- **和音の響き**（相対音感の入り口）
  和音を聴いて「明るい響き（メジャー）」か「暗い響き（マイナー）」かを当てます。

### メトロノーム
BPM 40〜208、2/3/4拍子に対応。タップしてテンポを入力することもできます。

### 練習の録画
カメラとマイクで練習の様子を録画し、端末にダウンロードできます（映像そのものはアプリ内には保存されません）。

### 日記・カレンダー
- 月表示のカレンダーで、練習した日に花丸「✿」が付きます。日付をタップするとその日の記録が下に開き、過去の日も編集できます。
- 各日の記録には「今日練習した曲（複数入力可）」と「自由記述」を残せます。
- 練習問題の正答数は**カテゴリ別の内訳と合計**が自動で記録されます（例：音階読み4/5問、耳で音当て2/3問、合計6/8問）。
- 1日に何度アプリを開いても、その日の記録は上書きされず積み上がり続けます。

## 使い方

1. このページ（GitHub Pagesで公開したURL）をiPhoneのSafariで開く
2. 共有ボタン → 「ホーム画面に追加」をタップ
3. ホーム画面にできたアイコンから開けば、以降はアプリのように起動できます

## データの保存について

- 練習記録・日記は、開いているブラウザの **localStorage**（端末内のブラウザ保存領域）に保存されます。
- そのため、**同じ端末・同じブラウザ（Safari）で開き続ける**ことで記録が積み重なります。
- 別の端末や別のブラウザからは記録は見えません（他人と共有されることもありません）。
- ブラウザの「サイトデータを消去」やSafariの履歴・データ削除を行うと、記録も消えるのでご注意ください。
- 家族や他の生徒と共用する場合は、記録が全員分混ざって保存される点にご注意ください（アカウント分けの機能はありません）。

## 音について

- クイズの音・メトロノーム・和音は、すべてブラウザ標準の Web Audio API でその場で生成しています。音声ファイルは含まれていません。
- iPhoneではマナーモードや音量設定の影響を受けます。また、ブラウザの仕様により最初の再生時に画面タップが必要な場合があります。

## 録画機能について

- カメラ・マイクへのアクセスは、`https://` で配信されているページ（GitHub Pagesなど）でのみ正常に動作します。
- 初回はブラウザからカメラ・マイクの利用許可を求められるので、「許可」を選んでください。
- 録画データはこのアプリの外（お使いの端末）にダウンロードして保存する形式です。写真アプリなどへの保存は、ダウンロード後にご自身で行ってください。

## 技術構成

- 素のHTML / CSS / JavaScriptのみで作られた単一ファイルのWebアプリです（フレームワーク不要）。
- サーバーやビルド手順は不要で、`index.html` をそのままGitHub Pagesなどで公開すれば動作します。
- 五線譜・鍵盤・音符・調号はSVG、音はWeb Audio APIで描画・生成しています。
- ファイルサイズは約50KBと軽量です。

## 補足

- 絶対音感トレーニングは本来かなり長期の訓練を要するものです。「耳で音当て」は音に親しむ入り口として気軽にお使いください。
- ♯・♭は「音階読み（♯あり）」「調・調号」で扱いますが、まず白鍵に慣れることを優先した設計になっています。
