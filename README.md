# おんぷノート 🎹

ピアノ初学者向けの練習手帳アプリです。インストール不要のWebアプリ（PWA）として、iPhoneのホーム画面から普通のアプリのように使えます。

とくに **読譜（楽譜をスラスラ読む力）の上達** に力を入れ、読む速さを測って伸びを見える化し、苦手な音に練習を集中できるようにしています。

## できること

### 練習メニュー
練習タブを開くとカテゴリを選ぶメニューが表示され、選んだ練習が始まります。「← 選び直す」でいつでもメニューに戻れます。

- **音階読み（白鍵）**
  五線譜に表示された音符を、画面下に**位置が固定された7つのドレミボタン（ド・レ・ミ・ファ・ソ・ラ・シ）**で答えます。選択肢を探す手間がないぶん、読む速さがそのまま測れます。正解すると音が鳴り、鍵盤のどの位置かが光ります。ト音記号／ヘ音記号を切り替えられます。
  - **出題音域は「適応／基本／ひろめ」**から選べます（既定は「適応」）。適応モードは**ドレミの3音から始まり、覚えた音から1音ずつ増えていく**ので、初めてでも無理なく進められます。
  - どのモードでも、**間違えやすい音・読むのが遅い音が多めに出題**されます（弱点集中）。画面下に今の到達度と「苦手な音」が表示されます。
- **音階読み（♯あり）**
  白鍵に加えて黒鍵の音（ド♯・レ♯・ファ♯・ソ♯・ラ♯）も出題されます（ドレミ4択）。異名同音は避け、♯表記に統一しています。
- **音階読み（鍵盤で）**
  五線譜の音符に対応するキーを、画面下のピアノ鍵盤（白鍵9個）をタップして答えます。「◀ 低い音 / 高い音 ▶」で音域を1オクターブずつ移動できます（全体はC3〜C5、ヘ音／ト音は自動切替）。
- **音列読み**（文脈読み）
  五線譜に**並んだ4つの音符**を、ドレミボタンで**左から順に**読みます。実際のフレーズのように音がつながって動くので、単音より一歩進んだ「読みの流れ」が鍛えられます。
- **音符の種類**
  全音符・二分音符・四分音符・八分音符のどれかを当てます。
- **調・調号**
  調号から、ハ長調・ト長調・ニ長調・イ長調・ヘ長調・変ロ長調・変ホ長調のどれかを当てます。
- **耳で音当て**（絶対音感の入り口）
  鳴った音がドレミのどれかを聴いて当てます。「もう一度きく」で何度でも聴き直せます。
- **和音の響き**（相対音感の入り口）
  和音を聴いて「明るい響き（メジャー）」か「暗い響き（マイナー）」かを当てます。
- **スピードテスト** ⏱
  読む速さを**定点観測**するテストです。毎回おなじ構成（ト音記号のドレミすべてを均等に・計21問）で出題順だけが変わるので、日をまたいでも公平に比べられます。終わると「平均◯秒／問・正答率」と、**前回より速くなったか**が表示されます。

### メトロノーム
BPM 40〜208、2/3/4拍子に対応。タップしてテンポを入力することもできます。

### 練習の録画
カメラとマイクで練習の様子を録画し、端末にダウンロードできます（映像そのものはアプリ内には保存されません）。

### 日記・カレンダー
- 月表示のカレンダーで、練習した日に花丸「✿」が付きます（問題を解く・スピードテストを受ける・曲や日記を記録する、のいずれかで付きます）。日付をタップするとその日の記録が下に開き、過去の日も編集できます。
- **連続記録（◯日連続）と、1日の目標問題数（デイリー目標）**を表示します。目標は5〜100問で変更できます。
- **読譜スピードの記録**として、スピードテストの平均反応時間の推移をグラフで確認できます（速いほど上）。
- 各日の記録には「今日練習した曲（複数入力可）」と「自由記述」を残せます。練習問題の正答数はカテゴリ別の内訳と合計が自動で記録されます。
- 1日に何度アプリを開いても、その日の記録は上書きされず積み上がります。日付は端末のローカルタイム基準です。

## 使い方

1. このページ（GitHub Pagesで公開したURL）をiPhoneのSafariで開く
2. 共有ボタン → 「ホーム画面に追加」をタップ
3. ホーム画面にできたアイコンから開けば、以降はアプリのように起動できます

## データの保存について

- 練習記録・日記・学習データは、開いているブラウザの **localStorage**（端末内のブラウザ保存領域）に保存されます。
- そのため、**同じ端末・同じブラウザ（Safari）で開き続ける**ことで記録が積み重なります。
- 別の端末や別のブラウザからは記録は見えません（他人と共有されることもありません）。
- ブラウザの「サイトデータを消去」やSafariの履歴・データ削除を行うと、記録も消えるのでご注意ください。
- 家族や他の生徒と共用する場合は、記録が全員分混ざって保存される点にご注意ください（アカウント分けの機能はありません）。

## 音について

- クイズの音・メトロノーム・和音は、すべてブラウザ標準の Web Audio API でその場で生成しています。音声ファイルは含まれていません。
- iPhoneではマナーモードや音量設定の影響を受けます。また、ブラウザの仕様により最初の再生時に画面タップが必要な場合があります。

## 録画機能について

- カメラ・マイクへのアクセスは、`https://` で配信されているページ（GitHub Pagesなど）でのみ正常に動作します。
- 初回はブラウザからカメラ・マイクの利用許可を求められるので、「許可」を選んでください。
- 録画データはこのアプリの外（お使いの端末）にダウンロードして保存する形式です。

## 技術構成

- 素のHTML / CSS / JavaScriptのみで作られた単一ファイルのWebアプリです（フレームワーク不要）。
- サーバーやビルド手順は不要で、`index.html` をそのままGitHub Pagesなどで公開すれば動作します。
- 五線譜・鍵盤・音符・調号・グラフはSVG、音はWeb Audio APIで描画・生成しています。

## 補足

- **上達の考え方**：このアプリが伸ばす「上達」は読譜（楽譜を速く正確に読む力）です。日々の練習は苦手な音に集中し、スピードテストで速さを測り、グラフで伸びを確認する——という流れで進みます。
- 絶対音感トレーニングは本来かなり長期の訓練を要するものです。「耳で音当て」は音に親しむ入り口として気軽にお使いください。
- ♯・♭は「音階読み（♯あり）」「音階読み（鍵盤で）」「調・調号」で扱いますが、固定ドレミパッド・弱点集中・適応・速度計測は白鍵を中心とした設計になっています。
